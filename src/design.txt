instance variable
-name ไว้เก็บชื่อของเครื่อง
-มี model ไว้เก็บรุ่นของ airpurifier
-มี serialNum ไว้เก็บรหัสของแต่ละเครื่อง
-มี color_case ไว้เก็บสีของเครื่อง
-มี pm เก็บค่าตัวเลขของค่า pm ในอากาศ
-_switch เก็บสถานะของเครื่องว่าเปิดหรือปิดอยู่
instance methods
-switchPushing เป็น ทำ method ในการเปลี่ยนสถานะของเครื่องเมื่อเรียกใช้จะแสดงการกดปุ่ม เมื่อกดปุ่มจากเครื่องที่ปิดอยู่จะเปิด เครื่องที่เปิดอยู่จะปิด
-sensorPm input เป็นค่าของ pm (เปรียบเสมือนเป็นการรับค่าจาก sensor)เป็น method ที่เก็บค่า pm ในอากาศไว้ในเครื่อง ในเครื่องกรองอากาศจะมี sensor สำหรับตรวจวัดค่า pm2.5
    methods sensorPm จะทำงานโดยเปรียบเสมือนการรับค่าจาก sensor ในครื่องแล้วเก็บค่าไว้เพื่อนำไปแสดงผลผ่าน display
-showModel เป็น method ที่ใช้ในการแสดงผลเกี่ยวกับตัวเครื่อง ได้แก่ name model serialNum color
-showDisplay เป็น method  ที่ใช้ในการแสดงผลบนหน้าจอของเครื่องกรองอากาศ ได้แก่ สถานะการเปิดปิดของเครื่อง ค่าปริมาณฝุ่น
class variable
-overallCount ใช้เก็บค่าของจำนวน airpurifier ที่มีการผลิตขึ้น
class methods
-setOverallCount เป็น method ในการเพิ่มค่าของจำนวนเครื่องที่ผลิตขึ้นมาในทุกๆครั้งที่เกิดการผลิต
-printOverAll เป็น method ในการแสดงจำนวนของเครื่องที่มีการผลิตทั้งหมด
